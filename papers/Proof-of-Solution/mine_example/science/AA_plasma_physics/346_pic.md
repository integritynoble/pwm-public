## Layer 1: Seeds -> Principle

### Governing Equation

```
m * dv_i/dt = q * (E(x_i) + v_i x B(x_i))      (Newton-Lorentz)
dx_i/dt = v_i                                     (particle push)
rho(x) = sum_i q * S(x - x_i)                    (charge deposit)
J(x) = sum_i q * v_i * S(x - x_i)               (current deposit)
nabla^2(phi) = -rho/eps0   or   Maxwell's eqs    (field solve)
```

### DAG Decomposition G = (V, A)

```
[S.particle] -> [∫.deposit] -> [L.poisson] -> [∂.time]

V = {S.particle, ∫.deposit, L.poisson, ∂.time}
A = {S.particle->∫.deposit, ∫.deposit->L.poisson, L.poisson->∂.time}
L_DAG = 3.0   Tier: standard (delta = 3)
```

### Well-Posedness Certificate

| Property | Status |
|----------|--------|
| Existence | YES -- particle trajectories well-defined for bounded E, B |
| Uniqueness | YES -- deterministic given initial particle distribution |
| Stability | CONDITIONAL -- CFL: omega_pe*dt < 2; dx < lambda_D |

Mismatch parameters: beam velocity v_b, density ratio, particles per cell ppc
