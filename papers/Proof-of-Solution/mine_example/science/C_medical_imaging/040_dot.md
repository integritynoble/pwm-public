## Layer 1: Seeds -> Principle

### Forward Model

```
-div(D(r)*grad(phi(r,t))) + mu_a(r)*phi(r,t) + (1/c)*dphi/dt = S(r,t)
y_d = phi(r_d, t) + n       (boundary measurements at detector d)

  G.pulse.laser --> K.scatter.diffuse --> S.boundary
```

### DAG Primitives

| Node | Primitive | Operation |
|------|-----------|-----------|
| G.pulse.laser | Generate | Laser pulse excitation |
| K.scatter.diffuse | Kernel | Diffuse scattering propagation |
| S.boundary | Sample | Boundary measurement |

### Well-Posedness

- Existence: YES -- diffusion equation has unique solution for given (mu_a, mu_s')
- Uniqueness: CONDITIONAL -- severely ill-posed; requires regularization
- Stability: POOR -- kappa ~ 200 (CW), kappa ~ 80 (FD multi-frequency)
- Mismatch: source position error (d_src), coupling coefficient (d_coup), boundary shape error (d_shape)

### Error Method

- Primary: relative error in mu_a reconstruction (%), PSNR (dB)
- Secondary: localization error (mm), contrast recovery (%)
- Convergence rate: q = 1.0 (Gauss-Newton), q = 0.5 (gradient descent)

### S1-S4 Gate Checks

| Gate | Check | Result |
|------|-------|--------|
| S1 | Source-detector geometry consistent with diffusion regime | PASS |
| S2 | Regularized inverse bounded despite high kappa | PASS |
| S3 | Gauss-Newton converges with Tikhonov regularization | PASS |
| S4 | Relative mu_a error <= 20% achievable for 2-inclusion phantom | PASS |
