## Layer 1: Seeds -> Principle

### Forward Model

```
y(theta, s) = integral f(s*cos(theta) - t*sin(theta), s*sin(theta) + t*cos(theta)) dt + n

  Π.radon.parallel --> S.angular.full
```

### DAG Primitives

| Node | Primitive | Operation |
|------|-----------|-----------|
| Π.radon.parallel | Radon transform | Parallel-beam line-integral projection |
| S.angular.full | Sample | Full-angle detector sampling |

### Well-Posedness

- Existence: YES -- Radon transform is injective for sufficient angles
- Uniqueness: YES -- Fourier Slice Theorem guarantees unique recovery
- Stability: CONDITIONAL -- kappa ~ 10 (full-view), kappa ~ 80 (sparse-view)
- Mismatch parameters: angular offset (d_theta), detector shift (ds), beam hardening (alpha_bh)

### Error Method

- Primary metric: PSNR (dB)
- Secondary: SSIM, RMSE (HU)
- Convergence rate: q = 2.0 (FBP analytic), q = 1.0 (iterative)

### S1-S4 Gate Checks

| Gate | Check | Result |
|------|-------|--------|
| S1 | Sinogram dimensions [N_theta x N_det] consistent with geometry | PASS |
| S2 | Full-angle sampling satisfies Nyquist; kappa bounded | PASS |
| S3 | FBP converges analytically; SART converges at O(1/k) | PASS |
| S4 | PSNR >= 30 dB achievable for standard phantoms | PASS |
