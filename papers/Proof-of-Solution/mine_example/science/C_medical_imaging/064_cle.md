## Layer 1: Seeds -> Principle

### Forward Model

```
y(x,y) = integral PSF_confocal(x,y,z) * F(x,y,z) dz + honeycomb + n
PSF_confocal = h_illum(x,y,z) * h_detect(x,y,z)   (confocal point spread)

  G.pulse.laser --> K.green.coherent --> S.fiber
```

### DAG Primitives

| Node | Primitive | Operation |
|------|-----------|-----------|
| G.pulse.laser | Generate | Laser pulse excitation |
| K.green.coherent | Kernel | Coherent optical Green’s function |
| S.fiber | Sample | Fiber-bundle spatial sampling |

### Well-Posedness

- Existence: YES -- fluorescence from focal plane is measurable
- Uniqueness: YES -- confocal sectioning provides single-plane image
- Stability: kappa ~ 10 (good contact), kappa ~ 30 (motion + poor contact)
- Mismatch: motion blur (d_motion), fiber pattern (d_fiber), photobleaching (d_bleach)

### Error Method

- Primary: PSNR (dB), SSIM for enhancement; mosaic quality metric
- Secondary: resolution improvement, frame registration accuracy
- Convergence rate: q = 2.0 (deconvolution), q = 1.5 (mosaicking)

### S1-S4 Gate Checks

| Gate | Check | Result |
|------|-------|--------|
| S1 | Fiber spacing and confocal depth consistent with resolution target | PASS |
| S2 | Fluorescence SNR sufficient for cellular-level imaging | PASS |
| S3 | Deconvolution and mosaicking converge | PASS |
| S4 | PSNR >= 25 dB for enhanced CLE frames | PASS |
