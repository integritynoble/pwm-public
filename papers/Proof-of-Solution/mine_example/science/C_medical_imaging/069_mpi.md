## Layer 1: Seeds -> Principle

### Forward Model

```
y(t) = d/dt integral c(r) * m(H(r,t)) dV + n
m(H) = M_s * L(mu_0*m*H / (k_B*T))   (Langevin function)
y = S * c + n     (S = system matrix from calibration, c = tracer concentration)

  G.pulse.rf --> N.pointwise --> ∫.temporal
```

### DAG Primitives

| Node | Primitive | Operation |
|------|-----------|-----------|
| G.pulse.rf | Generate | RF pulse excitation |
| N.pointwise | Nonlinear | Pointwise nonlinear response |
| ∫.temporal | Integrate | Temporal integration / accumulation |

### Well-Posedness

- Existence: YES -- SPIO particles generate harmonic signals
- Uniqueness: YES -- system matrix calibration provides unique mapping
- Stability: kappa ~ 20 (well-calibrated), kappa ~ 80 (partial system matrix)
- Mismatch: system matrix drift (d_SM), background signal (d_bg), particle relaxation (d_relax)

### Error Method

- Primary: PSNR (dB), NMSE of tracer concentration map
- Secondary: spatial resolution (mm), sensitivity (ng Fe)
- Convergence rate: q = 1.0 (Kaczmarz iterative), q = 2.0 (pseudoinverse)

### S1-S4 Gate Checks

| Gate | Check | Result |
|------|-------|--------|
| S1 | System matrix dimensions match drive field / grid | PASS |
| S2 | System matrix condition bounded with SVD thresholding | PASS |
| S3 | Kaczmarz iterative solver converges | PASS |
| S4 | PSNR >= 25 dB achievable for resolution phantom | PASS |
