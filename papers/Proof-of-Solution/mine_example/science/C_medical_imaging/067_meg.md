## Layer 1: Seeds -> Principle

### Forward Model

```
B(r_sensor) = (mu_0/4pi) * integral J(r') x (r_sensor - r') / |r_sensor - r'|^3 dV'
y = L * j + n     (L = lead field matrix, j = source currents)

  G.dipole --> K.green.em --> S.surface
```

### DAG Primitives

| Node | Primitive | Operation |
|------|-----------|-----------|
| G.dipole | Generate | Neural current dipole source |
| K.green.em | Kernel | Electromagnetic Green’s function |
| S.surface | Sample | Surface sensor array readout |

### Well-Posedness

- Existence: YES -- neural currents produce measurable magnetic fields
- Uniqueness: NO -- severely underdetermined (more sources than sensors)
- Stability: kappa ~ 300+ (highly ill-posed, requires strong regularization)
- Mismatch: head model error (d_head), coregistration error (d_coreg), sensor noise (d_sensor)

### Error Method

- Primary: source localization error (mm), dipole orientation error (deg)
- Secondary: spatial resolution (mm), source extent recovery
- Convergence rate: q = 1.0 (beamformer), q = 0.5 (sparse Bayesian)

### S1-S4 Gate Checks

| Gate | Check | Result |
|------|-------|--------|
| S1 | Sensor array and head model consistent with geometry | PASS |
| S2 | Regularized inverse (MNE/beamformer) bounded | PASS |
| S3 | MNE / LCMV beamformer converges | PASS |
| S4 | Localization error <= 10 mm for single dipole at SNR 5 | PASS |
