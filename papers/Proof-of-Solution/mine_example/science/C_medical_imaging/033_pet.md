## Layer 1: Seeds -> Principle

### Forward Model

```
y_i = sum_j c_ij * a_ij * f_j + s_i + r_i + n_i

  Π.line --> S.coincidence --> ∫.temporal
  c_ij = geometric system matrix, a_ij = attenuation factor
  s_i = scatter, r_i = randoms
```

### DAG Primitives

| Node | Primitive | Operation |
|------|-----------|-----------|
| Π.line | Line projection | Line-of-response integration |
| S.coincidence | Sample | Coincidence event binning |
| ∫.temporal | Integrate | Temporal integration / accumulation |

### Well-Posedness

- Existence: YES -- LOR projections are well-defined for non-negative tracer
- Uniqueness: YES -- sufficient angular coverage with attenuation correction
- Stability: CONDITIONAL -- kappa ~ 20 (standard count), kappa ~ 100 (low count)
- Mismatch: attenuation map error (d_mu), scatter estimate error (d_s), normalization error (d_n)

### Error Method

- Primary: PSNR (dB), RMSE (Bq/mL)
- Secondary: CNR (hot/cold lesions), bias (%), coefficient of variation
- Convergence rate: q = 1.0 (MLEM), q = 1.5 (OSEM)

### S1-S4 Gate Checks

| Gate | Check | Result |
|------|-------|--------|
| S1 | Sinogram dimensions match ring geometry; LOR count consistent | PASS |
| S2 | Count statistics sufficient for stable reconstruction | PASS |
| S3 | MLEM converges monotonically (log-likelihood increase) | PASS |
| S4 | PSNR >= 25 dB achievable for brain phantom | PASS |
