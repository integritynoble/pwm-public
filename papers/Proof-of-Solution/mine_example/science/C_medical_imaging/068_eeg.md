## Layer 1: Seeds -> Principle

### Forward Model

```
phi(r_sensor) = (1/4pi*sigma) * integral J(r') . grad(1/|r_sensor - r'|) dV'
y = L * j + n     (L = EEG lead field matrix, j = source currents)

  G.dipole --> K.green.em --> S.surface
```

### DAG Primitives

| Node | Primitive | Operation |
|------|-----------|-----------|
| G.dipole | Generate | Neural current dipole source |
| K.green.em | Kernel | Electromagnetic Green’s function |
| S.surface | Sample | Surface sensor array readout |

### Well-Posedness

- Existence: YES -- volume currents produce scalp potentials
- Uniqueness: NO -- severely underdetermined (Helmholtz decomposition: silent sources exist)
- Stability: kappa ~ 500+ (worse than MEG due to volume conduction)
- Mismatch: head conductivity error (d_sigma), electrode position error (d_pos), reference electrode (d_ref)

### Error Method

- Primary: source localization error (mm), dipole moment error (nAm)
- Secondary: spatial dispersion (mm), resolution metric
- Convergence rate: q = 1.0 (sLORETA), q = 0.5 (sparse methods)

### S1-S4 Gate Checks

| Gate | Check | Result |
|------|-------|--------|
| S1 | Electrode montage and head model consistent | PASS |
| S2 | Regularized inverse (sLORETA/beamformer) bounded | PASS |
| S3 | sLORETA / eLORETA converges | PASS |
| S4 | Localization error <= 15 mm for single dipole at SNR 3 | PASS |
