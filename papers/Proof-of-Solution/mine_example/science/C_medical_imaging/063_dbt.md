## Layer 1: Seeds -> Principle

### Forward Model

```
y(u,v,theta) = I0 * exp(- integral mu_breast(x,y,z) dl_theta ) + n
theta in [-25, +25] deg  (limited angular range)

  Π.radon.parallel --> S.angular
```

### DAG Primitives

| Node | Primitive | Operation |
|------|-----------|-----------|
| Π.radon.parallel | Radon transform | Parallel-beam line-integral projection |
| S.angular | Sample | Angular detector sampling |

### Well-Posedness

- Existence: YES -- projections measurable for non-negative mu
- Uniqueness: CONDITIONAL -- limited angle causes incomplete data; regularization needed
- Stability: kappa ~ 30 (standard 15 views), kappa ~ 80 (sparse-view)
- Mismatch: tube motion blur (d_blur), scatter (d_scatter), detector lag (d_lag)

### Error Method

- Primary: PSNR (dB), SSIM of reconstructed slices
- Secondary: lesion detectability (AUC), in-plane vs. depth resolution
- Convergence rate: q = 1.5 (shift-and-add), q = 1.0 (iterative SART)

### S1-S4 Gate Checks

| Gate | Check | Result |
|------|-------|--------|
| S1 | Projection angles and detector geometry consistent | PASS |
| S2 | Regularized limited-angle inversion bounded | PASS |
| S3 | SART or iterative TV converges | PASS |
| S4 | PSNR >= 25 dB achievable for ACR phantom | PASS |
