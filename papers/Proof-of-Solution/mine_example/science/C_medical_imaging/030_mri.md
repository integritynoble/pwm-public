## Layer 1: Seeds -> Principle

### Forward Model

```
y(kx,ky) = integral integral m(x,y) * exp(-i*2*pi*(kx*x + ky*y)) dx dy + n

  L.mix.coil --> F.dft --> S.kspace.cartesian
```

### DAG Primitives

| Node | Primitive | Operation |
|------|-----------|-----------|
| L.mix.coil | Linear mix | Multi-coil sensitivity weighting |
| F.dft | Fourier | Discrete Fourier transform encoding |
| S.kspace.cartesian | Sample | Cartesian k-space readout |

### Well-Posedness

- Existence: YES -- Fourier transform is bijective for full sampling
- Uniqueness: YES -- fully sampled k-space uniquely determines image
- Stability: CONDITIONAL -- kappa ~ 5 (full), kappa ~ 50 (4x accelerated)
- Mismatch parameters: B0 inhomogeneity (dB0), coil sensitivity error (dC), k-space trajectory deviation (dk)

### Error Method

- Primary: PSNR (dB), SSIM
- Secondary: NMSE
- Convergence rate: q = 2.0 (IFFT analytic), q = 1.0 (CS iterative)

### S1-S4 Gate Checks

| Gate | Check | Result |
|------|-------|--------|
| S1 | k-space samples consistent with image grid and coil count | PASS |
| S2 | Acceleration factor within CS recoverability regime | PASS |
| S3 | IFFT (full) or ADMM (accel.) converges with known bounds | PASS |
| S4 | PSNR >= 30 dB achievable for standard brain datasets | PASS |
