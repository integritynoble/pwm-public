## Layer 1: Seeds -> Principle

### Forward Model

```
div(sigma(r) * grad(phi(r))) = 0     (Laplace equation)
y_ij = phi(r_j) | I injected at electrode i + n
Neumann-to-Dirichlet map: Lambda_sigma(I) = V_boundary

  G.current --> L.dense --> S.boundary
```

### DAG Primitives

| Node | Primitive | Operation |
|------|-----------|-----------|
| G.current | Generate | Current injection pattern |
| L.dense | Linear dense | Dense forward-model matrix |
| S.boundary | Sample | Boundary measurement |

### Well-Posedness

- Existence: YES -- boundary voltages uniquely determined by sigma
- Uniqueness: YES -- Calderon problem: Lambda_sigma determines sigma (2D, smooth)
- Stability: SEVERELY ILL-POSED -- kappa ~ 500+ (exponential instability)
- Mismatch: electrode contact impedance (d_contact), boundary shape error (d_shape), electrode position error (d_pos)

### Error Method

- Primary: conductivity RMSE (%), spatial resolution (mm)
- Secondary: contrast recovery (%), position error of inclusions
- Convergence rate: q = 0.5 (Gauss-Newton with heavy regularization)

### S1-S4 Gate Checks

| Gate | Check | Result |
|------|-------|--------|
| S1 | Electrode count and current patterns consistent | PASS |
| S2 | Regularized inverse bounded despite extreme kappa | PASS |
| S3 | Gauss-Newton with Tikhonov/TV converges | PASS |
| S4 | Conductivity RMSE <= 25% for 2-inclusion tank phantom | PASS |
