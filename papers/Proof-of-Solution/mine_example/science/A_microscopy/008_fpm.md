## Layer 1 — Seeds → Principle

```
┌──────────────────────────────────────────────────────────────────┐
│  FPM  P = (E, G, W, C)   Principle #8                         │
├────────┬─────────────────────────────────────────────────────────┤
│   E    │ y_k(r) = |F⁻¹{P(u) · F{o(r)} shifted by u_k}|² + n │
│        │ o(r) = complex transmittance; P(u) = pupil function  │
│        │ u_k = LED-dependent shift in Fourier space (k=1..N)  │
│        │ Inverse: recover amplitude+phase of o at high NA_syn │
├────────┼─────────────────────────────────────────────────────────┤
│   G    │ [G.structured.led]──→[K.psf.ctf]──→[N.pointwise.abs2]  │
│        │  Source(LED-array)  CTF-filter  Intensity(|·|²)        │
│        │ V={G,K,N}  A={G-->K, K-->N}   L_DAG=4.0              │
├────────┼─────────────────────────────────────────────────────────┤
│   W    │ Existence: YES (overlapping Fourier patches cover)    │
│        │ Uniqueness: YES with > 60% overlap in Fourier space   │
│        │ Stability: κ ≈ 40 (good overlap); κ ≈ 200 (sparse)  │
│        │ Mismatch: LED position error, aberrations, vignetting │
├────────┼─────────────────────────────────────────────────────────┤
│   C    │ e = PSNR (amplitude), phase_RMSE (rad)               │
│        │ q = 1.5 (alternating projection convergence)        │
│        │ T = {residual_norm, phase_error, NA_syn achieved}    │
└────────┴─────────────────────────────────────────────────────────┘
```
