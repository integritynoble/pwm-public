## Layer 1 — Seeds → Principle

```
┌──────────────────────────────────────────────────────────────────┐
│  DIC  P = (E, G, W, C)   Principle #18                        │
├────────┬─────────────────────────────────────────────────────────┤
│   E    │ y(r) = I_0 · [1 + cos(Δφ(r) + β)]                   │
│        │ Δφ = (2π/λ)·Δn·∂t/∂s · d (phase gradient along shear)│
│        │ β = bias retardation; d = Nomarski shear distance     │
│        │ Inverse: integrate phase gradient → recover φ(r)     │
├────────┼─────────────────────────────────────────────────────────┤
│   G    │ [K.psf] ──→ [∂.finite] ──→ [N.pointwise.cos]          │
│        │  PSF-filter  Gradient(shear)  Nonlinear(cos-interference)│

├────────┼─────────────────────────────────────────────────────────┤
│   W    │ Existence: YES (gradient is measurable via intensity) │
│        │ Uniqueness: YES up to constant (integration constant) │
│        │ Stability: κ ≈ 15 (optimal bias β=π/4)              │
│        │ Mismatch: shear angle θ, bias β, polarizer alignment  │
```
