## Layer 1 — Seeds → Principle

```
┌──────────────────────────────────────────────────────────────────┐
│  AFM   P = (E, G, W, C)   Principle #162                        │
├────────┬─────────────────────────────────────────────────────────┤
│   E    │ y(x,y) = [T(x,y) ⊕ S(x,y)] + n(x,y)                  │
│        │ T = tip geometry, S = true surface topography           │
│        │ ⊕ = dilation (morphological convolution)               │
│        │ Inverse: erode measured image by tip to recover S       │
├────────┼─────────────────────────────────────────────────────────┤
│   G    │ [K.morphological] --> [S.raster] --> [N.force]           │
│        │  TipSurface  RasterScan  ForceDetect                   │
│        │ V={K.morphological, S.raster, N.force}  A={K.morphological-->S.raster, S.raster-->N.force}   L_DAG=1.5│
├────────┼─────────────────────────────────────────────────────────┤
│   W    │ Existence: YES (tip-sample interaction is local)       │
│        │ Uniqueness: YES for features wider than tip apex        │
│        │ Stability: κ ≈ 5 (tapping), κ ≈ 15 (contact)         │
│        │ Mismatch: tip wear, drift, piezo nonlinearity           │
├────────┼─────────────────────────────────────────────────────────┤
│   C    │ e = RMSE_z (nm), lateral_resolution (nm)               │
│        │ q = 2.0 (erosion convergence for known tip shape)    │
│        │ T = {RMSE_z, lat_res_nm, roughness_Ra, drift_rate}     │
└────────┴─────────────────────────────────────────────────────────┘
```
