│  MFM   P = (E, G, W, C)   Principle #165                        │

│   W    │ Existence: YES (magnetic stray field decays as 1/r³)   │
│        │ Uniqueness: YES for in-plane features > lift height     │
│        │ Stability: κ ≈ 10 (lift 50 nm), κ ≈ 30 (lift 20 nm)  │
│        │ Mismatch: electrostatic cross-talk, tip magnetization   │
