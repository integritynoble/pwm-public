## Layer 1 — Seeds → Principle

```
┌──────────────────────────────────────────────────────────────────┐
│  SINGLE-PIXEL CAMERA  P = (E, G, W, C)   Principle #26        │
├────────┬─────────────────────────────────────────────────────────┤
│   E    │ y_k = <φ_k, f> + n_k,  k = 1..M                     │
│        │ φ_k = DMD pattern (binary/Hadamard/random); f = image │
│        │ M ≪ N (compressive: measure M < N = H×W values)      │
│        │ Inverse: recover f ∈ R^N from M measurements y       │
├────────┼─────────────────────────────────────────────────────────┤
│   G    │ [M] ──→ [Σ] ──→ [D]                                   │
│        │ Modulate  Sum(inner product)  Detect(single detector)  │
│        │ V={M,Σ,D}  A={M→Σ, Σ→D}   L_DAG=2.5                  │
├────────┼─────────────────────────────────────────────────────────┤
│   W    │ Existence: YES (CS theory, M ≥ O(s·log N))           │
│        │ Uniqueness: YES under sparsity s in some basis        │
│        │ Stability: κ ≈ 30 (Hadamard); κ ≈ 80 (random binary) │
│        │ Mismatch: pattern misalignment, detector nonlinearity  │
├────────┼─────────────────────────────────────────────────────────┤
│   C    │ e = PSNR, SSIM                                        │
│        │ q = 1.5 (ISTA/FISTA convergence rate)               │
│        │ T = {residual_norm, fitted_rate, compression_ratio}   │
└────────┴─────────────────────────────────────────────────────────┘
```
