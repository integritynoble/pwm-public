## Layer 1 — Principle

### Governing Equation

EFIE: n̂×∫_S G(r,r')·J(r') dS' = n̂×E_inc(r), r ∈ S
G(r,r') = (e^{−jk|r−r'|})/(4π|r−r'|) free-space Green's function
Discretize J with RWG basis → Z·I = V (dense linear system)

### DAG

```
[K.green.em] --> [L.dense] --> [B.surface]
Green-function-kernel  dense-system-solve  surface-BC
```

V={K.green.em,L.dense,B.surface}  L_DAG=3.0

### Well-Posedness

| Property | Status | Justification |
|----------|--------|---------------|
| Existence | YES | Fredholm integral equation of the first kind with regularization |
| Uniqueness | CONDITIONAL | EFIE has spurious resonances; CFIE resolves uniqueness |
| Stability | CONDITIONAL | κ(Z) grows with frequency; preconditioning needed |

Mismatch: mesh quality, basis function order, integration quadrature, near-field singularity treatment

### Error Method

e = RCS error (dB), relative current error ‖J_num − J_ref‖/‖J_ref‖
q = 2.0 (RWG linear), q = 3.0 (higher-order basis)
