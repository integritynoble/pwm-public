## Layer 1 — Principle

### Governing Equation

σ(θ,φ) = lim_{r→∞} 4πr² |E_s|²/|E_i|²
Physical optics: J_PO = 2n̂×H_inc on lit surface, J = 0 on shadow
Full-wave: solve EFIE/MFIE → extract far-field → compute σ

### DAG

```
[K.green.em] --> [L.dense] --> [B.surface] --> [∫.angular]
Green-kernel  impedance-solve  surface-BC  bistatic-integral
```

V={K.green.em,L.dense,B.surface,∫.angular}  L_DAG=3.0

### Well-Posedness

| Property | Status | Justification |
|----------|--------|---------------|
| Existence | YES | Bounded scattering cross section for finite targets |
| Uniqueness | YES | Far field uniquely determined by surface current |
| Stability | CONDITIONAL | High-frequency regime: PO adequate; resonance: full-wave needed |

Mismatch: geometry discretization, edge diffraction, creeping waves, material coating

### Error Method

e = RCS error (dBsm), angular pattern correlation
q = 1.0 (PO), q = 2.0 (MoM), convergence with frequency/mesh
